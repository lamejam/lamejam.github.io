Lame Jam

Learn. Game. Jam.

The Lame Jam is a beginner friendly game jam suitable for absolute beginners or game jam veterans fully equipped with workshops and mentors on all things game development and design.

Herzberg Labs

November 24 - 26, 2017

What is It?

Whether you're new to game jams or you're a seasoned veteran, Lame Jam is dedicated to providing a large game jam experience in a smaller, more beginner friendly environment.

If you're new to game development or design, we'll be hosting multiple short talks to hone your skills and build new ones. Snacks will be provided as well as a pancake breakfast on the Sunday morning.

When is It?

The event will be held 4:00PM Friday, November 24th - 4:00PM Sunday, November 26th with short, optional talks running all day Saturday and introductory talks on Friday.

Where is It?

The event will be happening all weekend in the Herzberg labs at Carleton University with some optional outings to on campus restaurants.
